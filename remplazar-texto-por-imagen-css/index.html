Remplazar texto por imagen (CSS)

Existen varias formas de remplazar texto con una imagen en css: image replacement museum pero la mas común de todas ellas es la siguiente:

.hide-text{
  text-indent: -9999px;
}

El problema con este método, es que le estamos indicando al navegador que cree un cuadro vacío de 9999px, el cual nosotros no vemos, pero esta siendo renderizado por el navegador, esto puede resultar en un problema de rendimiento, mas que todo en aparatos no tan recientes eg. (Ipad 1)

El método que utilizo ahora es la siguiente:

.hide-text{
  text-indent: 100%;
  white-space: nowrap;
  overflow: hidden;
}

Con este método, le estamos indicando la navegador que indente el texto 100%, esto sumado con el overflow: hidden; que oculta todo lo que sobrepase el ancho determinado, termina ocultando el texto y mejoramos el rendimiento, ya que ahora el navegador no tiene que renderizar un cuadro de 9999px.

Si por alguna razon, mas que todo en inputs de tipo submit, vemos que no esta funcionando al 100%, bastara con darle al input un padding:0; para se funcione correctamente, esto pasa porque el navegador le agrega padding por defecto.
